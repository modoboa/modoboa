# Requirements

You will need a Server to perform well with at least:

- CPU: 2
- RAM: 2GB
- Disk: 10GB
- Python 3.8+
- Postgres 12+ or MariaDB 10.4+ or MySQL 8+ or sqlite3 or LDAP
